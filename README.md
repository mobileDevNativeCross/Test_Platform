# Test_Platform

to run server:
```
npm i
npm start
```
it will launch server on http://localhost:3000  


to run test: 
```
npm i 
npm run test
```
it will run test and show results

to run doc:
```
npm i
npm run doc
npm run docServer
```
it will run small server on http://localhost:3002 with docs

to run client:
```
cd client/
npm i
npm start
```
it will run and open tab on browser http://localhost:3001
